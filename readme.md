# Ferramentas
- Nodejs
- Typescript
- Reactjs
- Sass
- Eslint
- Docker
- Express
- Cors
- Axios

# Passo a Passo
<h3> Requisitos </h3> 
- Nodejs instalado globalmente na maquina
- Docker instalado globalmente na maquina

<h3> Clone </h4> 
- git clone <url-projeto>

<h3> Front </h3> 
- execute npm i -l para instalar todas as dependencias 
<h3> Api </h3> 
- execute npm i -l para instalar todas as dependencias

# Execução
<h3> Execute o docker em ambas aplicações </h3> 
- npm run up
